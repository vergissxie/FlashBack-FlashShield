# FlashShield 演示操作手册

## 1. 当前推荐演示流程

1. 打开前端页面并连接钱包
2. 保持默认参数，或把策略 ID 改成一个新的值
3. 点击 `1. 开启演示仓位`
4. 点击 `2. 触发接近清算`
5. 等待价格滑落动画结束，再等待 Reactive 回调完成
6. 如有必要，点击 `刷新状态`

## 2. 你现在会看到什么

- 点击 `连接钱包` 后，页面才算“本页已连接”
- 点击 `1. 开启演示仓位` 后：
  - A 链会创建新策略
  - 页面显示“已开仓”
- 点击 `2. 触发接近清算` 后：
  - 钱包先要求你签名这次 A 链写入
  - 签名发出交易后，页面播放价格滑落动画
  - 动画结束后，页面刷新 A 链结果
  - 稍后 Reactive 将结果带到 B 链

## 3. 当前页面修复点

- 不会再把旧策略的保护结果误显示成当前策略结果
- 如果当前策略还没在 B 链保护成功，右侧和底部会显示 `未触发`
- 如果当前策略根本还没开仓，点击 `刷新状态` 也不会报错，只会显示空状态
